Detailed Description

template<typename T>
class dmdyn::SmartPList< T >

particle field list model

buffer model:

|0|1|2|...|10|11|12|...|20|21|...|29|30| | | | | V V V V start _len-1 _len+_vlen-1 buffer size-1 =1.5(_len+_vlen)-1;
